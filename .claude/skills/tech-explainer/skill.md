# Tech Explainer — 기술 원리 풀어쓰기 스킬

## 목적
AI 기술의 복잡한 개념을 IT 신입사원도 이해할 수 있도록 풀어서 설명.

## 설명 프레임워크

### 1. 한줄 정의
"[용어]란 [가장 핵심적인 것]을 [하는 것]입니다."

### 2. 일상 비유
```markdown
> **쉽게 이해하기: [용어]**
>
> [일상생활 비유로 설명]
>
> 예를 들어, [구체적 상황]에서 [이렇게 동작하는 것]과 같습니다.
```

### 3. 단계별 원리
```markdown
**[용어]는 이렇게 동작합니다:**

1️⃣ **[단계 1 제목]**: [설명]
2️⃣ **[단계 2 제목]**: [설명]
3️⃣ **[단계 3 제목]**: [설명]
```

### 4. 왜 중요한가 (So What?)
- 이 기술이 없었다면? → 이 기술 덕분에?
- 실제 서비스에서의 적용 예시

### 5. 더 알고 싶다면
- 추천 논문/블로그 1-2개 링크

## 설명 난이도 기준 (IT 신입사원)
- **알고 있다고 가정**: 프로그래밍 기초, 웹/앱 개발 기초, API 개념
- **모를 수 있다고 가정**: 딥러닝, 자연어처리, 강화학습, 분산처리
- **반드시 설명**: Transformer, Attention, 토큰, 임베딩, 파인튜닝 등

## 주의사항
- 지나치게 단순화하여 틀린 설명이 되지 않도록
- "정확하지는 않지만 이해를 위해" 식의 단서 대신, 정확하면서도 쉬운 표현 찾기
- 기술적 깊이는 "왜 이 방식이 좋은가"를 이해할 수 있는 수준까지
