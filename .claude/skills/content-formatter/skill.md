## 스타일 가이드

### 제목 계층
- `#` : 문서 제목 (1개만)
- `##` : 챕터 제목
- `###` : 섹션 제목
- `####` : 서브섹션 제목

### 강조 요소
- **굵게**: 핵심 용어 첫 등장, 중요 수치
- *기울임*: 인용, 강조
- `코드`: 모델명, 도구명, 기술 용어
- > 인용: 쉬운 설명 박스, 비즈니스 인사이트

### 표 스타일
- 헤더 행은 항상 포함
- 열 정렬: 텍스트는 좌측, 수치는 우측
- 5열 이상 시 가로 스크롤 고려하여 축약

### 이모지 사용 규칙
- 챕터 시작 요약 박스에서만 제한적 사용
- 본문에는 사용하지 않음
- 허용 이모지: 💡(인사이트), 📌(핵심), ⚡(속도/변화), 🔑(핵심 개념)
