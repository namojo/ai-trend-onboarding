# Ch4. AI가 바꾸는 세상

> **한줄 요약**: AI는 더 이상 연구실의 기술이 아니다 -- 검색, 코딩, 디자인, 영상, 업무 자동화까지, 이미 당신의 일상 곳곳에 침투해 있으며 수천억 달러의 산업을 만들어내고 있다.

월요일 아침, 당신이 출근합니다. Slack을 열면 AI가 주말 동안 쌓인 200개의 메시지를 세 줄로 요약해줍니다. 메일함의 긴 리포트는 Notion AI가 핵심만 뽑아놓았고, 오후 회의 자료는 v0에 "분기 매출 대시보드 만들어줘"라고 한 줄 입력하면 React 컴포넌트가 뚝딱 나옵니다. 퇴근 전 궁금한 게 생기면 Perplexity에 물어봅니다. 출처까지 달린 답변이 3초 만에 돌아옵니다.

이것이 2026년 3월 현재, 실제로 가능한 일상입니다.

이 챕터에서는 AI가 구체적으로 어떤 서비스와 제품으로 세상에 나와 있는지, 그리고 그것이 비즈니스에 어떤 숫자를 만들어내고 있는지를 살펴봅니다.

## 4.1 분야별 AI 서비스 지도

AI 서비스를 한 가지로 묶어 말하기는 어렵습니다. 검색을 바꾸는 AI, 코드를 대신 써주는 AI, 디자인을 뚝딱 만드는 AI, 영상을 편집하는 AI, 회사 업무를 자동화하는 AI까지 -- 각 분야마다 이미 치열한 경쟁이 벌어지고 있습니다. 자동차 산업에 비유하면, 범용 모델(GPT, Claude, Gemini 등)은 "엔진"이고 여기서 소개하는 서비스들은 그 엔진을 탑재한 "완성차"입니다.

### A. AI 검색 -- "구글링의 종말?"

10개의 파란 링크를 훑어보던 시대가 저물고 있습니다. AI 검색은 질문을 던지면 여러 웹 페이지를 종합해서 하나의 답변으로 돌려줍니다. 마치 개인 리서치 비서가 붙어 있는 셈입니다.

| 서비스 | 제공사 | 핵심 특징 | 규모 |
|--------|--------|-----------|------|
| **Perplexity** | Perplexity AI | 출처 기반 답변, 실시간 웹 검색 | 월 4,500만 사용자 (2025 중반) |
| **ChatGPT Search (Atlas)** | OpenAI | 대화형 검색, 웹 브라우징 통합 | ChatGPT 내장 |
| **Gemini** | Google | Google 검색 통합, 멀티모달 | Google 생태계 전체 연동 |
| **Arc Search** | The Browser Company | 브라우저 네이티브 AI 검색 | - |

주목할 숫자가 있습니다. ChatGPT의 AI 어시스턴트 시장 점유율이 87%에서 68%로 하락한 반면, Gemini가 18.2%로 급성장했습니다. Google이 Android에 Gemini를 기본 탑재하고 검색 생태계와 통합한 효과입니다. "AI 검색" 시장은 이제 한 회사가 독점하는 것이 아니라, 용도와 생태계에 따라 분화되고 있습니다.

### B. AI 코딩 도구 -- 개발자의 새로운 짝꿍

"바이브 코딩(Vibe Coding)"이라는 신조어가 등장할 만큼, AI와 함께 코딩하는 것이 2025~2026년의 표준이 되었습니다. 이 분야는 가장 폭발적으로 성장한 AI 서비스 카테고리입니다.

| 서비스 | 형태 | 기반 모델 | 가격 | 핵심 강점 |
|--------|------|-----------|------|-----------|
| **Cursor** | IDE (VS Code 포크) | Claude, GPT, 자체 모델 혼합 | 무료 / $19/월 | 올인원 IDE 경험, 자동완성+에이전트 |
| **GitHub Copilot** | IDE 확장 | GPT-5 계열, Claude | $10~39/월 | 엔터프라이즈 표준, 코드 리뷰 자동화 |
| **Windsurf** | IDE | 자체 모델 + Claude/GPT | 무료 / $15/월 | 인라인 완성 품질, 저렴한 가격 |
| **Claude Code** | CLI 도구 | Claude Sonnet/Opus | 무료 / $25/월 | 터미널 기반, 코드베이스 전체 이해 |

> **숫자로 보는 성장**: Cursor는 출시 24개월 만에 ARR 10억 달러를 돌파하며 B2B SaaS 역사상 최고속 성장을 기록했습니다. 2026년 3월 기준 ARR은 20억 달러를 넘어섰고, 기업가치는 293억 달러에 달합니다.

바이브 코딩의 상세한 실습 방법과 도구 비교는 Ch5에서 깊이 있게 다룹니다.

### C. AI 에이전트 -- 대화에서 행동으로

AI 챗봇에게 "이것 좀 해줘"라고 말하면, 진짜로 해주는 시대가 왔습니다. AI 에이전트는 단순히 텍스트를 생성하는 것을 넘어, 환경을 탐색하고, 도구를 사용하고, 멀티스텝 작업을 자율적으로 수행합니다.

상상해보세요. "이 GitHub 레포에서 버그를 찾아서 PR을 올려줘"라고 하면, Devin이 혼자서 코드를 분석하고, 수정하고, 테스트를 돌리고, Pull Request까지 생성합니다.

| 서비스 | 제공사 | 출시 | 핵심 특징 |
|--------|--------|------|-----------|
| **Devin** | Cognition | 2024 (프리뷰) → 2025 (정식) | 완전 자율 코딩 에이전트 -- 환경 구축부터 배포까지 |
| **Manus** | Manus AI (중국) | 2025.03 | GAIA 벤치마크 기록 경신, 72시간 내 18만 사용자 확보 |
| **AutoGPT** | 오픈소스 커뮤니티 | 2023~ | 자율 에이전트 프레임워크, 프로토타이핑용 |
| **CrewAI** | CrewAI | 2024~ | 멀티에이전트 팀 구성, Fortune 500의 40% 채택 |
| **LangChain/LangGraph** | LangChain | 2023~ | 에이전트 프레임워크, GitHub 87K 스타 |

이 시장의 규모는 2025~2026년 37억 달러에서 73.8억 달러로 2년 만에 2배 성장했으며, 2032년에는 1,036억 달러 규모로 전망됩니다.

> **주목할 인수**: 2025년 12월, Meta가 Manus를 20~30억 달러에 인수한다고 발표했습니다. 빅테크가 AI 에이전트 역량을 확보하기 위해 얼마나 공격적으로 움직이는지를 보여주는 사건입니다.

### E. 업무 자동화 -- AI가 사무실에 출근하다

가장 많은 직장인이 매일 접하게 될 AI 서비스는 바로 업무 자동화 도구입니다. Word, Excel, PPT, Slack, Notion -- 이미 쓰고 있는 도구 안에 AI가 들어왔습니다.

| 서비스 | 제공사 | 핵심 특징 |
|--------|--------|-----------|
| **Microsoft 365 Copilot** | Microsoft | Word/Excel/PPT/Teams AI 통합, 에이전트 스토어 |
| **Notion AI** | Notion | 문서 요약/작성/번역, 위키 Q&A |
| **Slack AI** | Salesforce | 채널 요약, 스레드 검색, 자동 응답 |
| **Zapier AI** | Zapier | 자연어로 자동화 워크플로 구축 |

"지난주 마케팅 채널에서 무슨 논의가 있었지?"라고 Slack AI에게 물으면, 100개의 메시지를 읽을 필요 없이 핵심 결정 사항과 액션 아이템을 정리해서 보여줍니다. Microsoft 365 Copilot은 "지난 분기 매출 데이터로 경영진 보고용 PPT 만들어줘"라는 한마디로 초안을 생성합니다.

### F. AI 하드웨어 -- 아직은 시기상조

모든 AI 서비스가 성공한 것은 아닙니다. "스마트폰을 대체하겠다"며 등장한 AI 전용 하드웨어들은 뼈아픈 실패를 맛보았습니다.

| 제품 | 가격 | 결과 | 교훈 |
|------|------|------|------|
| **Humane AI Pin** | $699 + $24/월 | 2025.02 HP가 1.16억 달러에 인수, 서버 종료로 기기 벽돌화 | 스마트폰 대체는 시기상조 |
| **Rabbit R1** | $199 | 10만 대 사전예약 후 5개월 만에 활성 사용자 5,000명 (95% 이탈) | 창업자 본인이 "너무 일찍 출시"했다고 인정 |

두 제품 모두 기술적 결함이 아니라 더 근본적인 문제에 부딪혔습니다. "스마트폰이 곧 같은 기능을 제공할 텐데, 소비자가 별도 기기를 살 이유가 없다"는 것이 전문가들의 공통된 분석입니다. 새로운 기술이 성공하려면 기존 습관을 이길 만큼의 압도적 이점이 필요하다는 점을 보여주는 사례입니다.

## 4.2 엔진과 완성차 -- 모델-서비스 관계도

앞서 살펴본 서비스들은 어떤 AI 모델 위에서 돌아가고 있을까요? 현대 AI 산업의 구조를 이해하는 핵심은 바로 이 "엔진-완성차" 관계입니다.

```
┌─────────────────────────────────────────────────────┐
│                    응용 서비스 (완성차)                │
│  Cursor  Perplexity  Notion AI  v0  Slack AI  ...  │
└────────────────────┬────────────────────────────────┘
                     │ API 호출 / 파인튜닝
┌────────────────────▼────────────────────────────────┐
│                    범용 모델 (엔진)                    │
│     GPT-5    Claude Opus/Sonnet    Gemini 3   ...   │
└─────────────────────────────────────────────────────┘
```

흥미로운 점은 대부분의 서비스가 하나의 모델에만 의존하지 않는다는 것입니다. Cursor는 Claude, GPT, 자체 모델을 혼합 사용하고, Microsoft 365 Copilot도 GPT-5에 더해 Claude를 "Cowork"라는 이름으로 통합했습니다. 마치 자동차 제조사가 상황에 따라 다른 엔진을 탑재하는 것과 같습니다.

반면, ChatGPT(Atlas), Gemini, Claude Code처럼 모델 제공사가 직접 서비스를 운영하는 경우도 있습니다. 이는 엔진 제조사가 직접 완성차를 만드는 셈입니다.

| 서비스 | 기반 모델 | 활용 방식 | 카테고리 |
|--------|-----------|-----------|----------|
| Cursor | Claude, GPT-5, 자체 모델 | API + 파인튜닝 | 코딩 |
| GitHub Copilot | GPT-5, Claude | API + 파인튜닝 | 코딩 |
| Perplexity | GPT, Claude, 자체 모델 | API + RAG 파이프라인 | 검색 |
| ChatGPT (Atlas) | GPT-5 계열 | 자사 모델 직접 통합 | 검색/대화 |
| Gemini | Gemini 3 Pro | 자사 모델 직접 통합 | 검색/대화 |
| Devin | Claude, GPT + 자체 모델 | API + 에이전트 로직 | 에이전트 |
| Manus | 다중 모델 조합 | API + 에이전트 오케스트레이션 | 에이전트 |
| MS 365 Copilot | GPT-5 + Claude | API + 깊은 제품 통합 | 업무 자동화 |
| Notion AI | Claude + 자체 모델 | API | 업무 자동화 |
| Slack AI | Claude | API | 업무 자동화 |
| CapCut AI | 자체 모델 (ByteDance) | 자체 모델 통합 | 영상 편집 |

AI가 만들어내는 비즈니스의 규모는 놀랍습니다. 몇 년 전까지 연구 프로젝트에 가까웠던 AI 기업들이 이제는 수백억 달러의 기업가치를 인정받고 있습니다.

## 4.4 지금 AI 세상을 움직이는 다섯 가지 흐름

개별 서비스와 숫자 너머에, AI 산업 전체를 관통하는 큰 흐름이 있습니다.

<!-- IMAGE: [다섯 가지 트렌드를 나타내는 인포그래픽. 중앙에 "AI 메가트렌드 2025-2026"이라는 타이틀을 두고, 다섯 방향으로 뻗어나가는 형태. 각 방향에 (1) 오픈소스의 반격 -- 가격 하락 화살표, (2) 에이전트의 시대 -- 로봇 아이콘, (3) 멀티모달 일상화 -- 텍스트/이미지/영상/음성 아이콘, (4) 비용 급락 -- 1/20 가격표, (5) 바이브 코딩 -- 노트북+주말 아이콘이 배치된다.] -->

### 1. 오픈소스의 반격

"AI 개발에는 수십억 달러가 필요하다"는 통념을 깨뜨린 것은 DeepSeek-V3였습니다. 최종 학습 비용 약 557만 달러로 프론티어급 성능을 달성했습니다. LLaMA 4, Grok, Qwen 3 등 오픈소스 모델들이 상용 모델과 대등한 성능을 보여주면서, AI의 민주화가 가속되고 있습니다.

### 2. 에이전트의 시대

2025년은 AI가 "대화"에서 "행동"으로 전환된 해입니다. Devin, Manus, Claude Code 등이 단순한 응답을 넘어 실제 작업을 자율적으로 수행하는 에이전트로 진화했습니다. Microsoft는 Agent 365라는 엔터프라이즈 에이전트 플랫폼까지 내놓았습니다. "시키면 답변해주는 AI"에서 "시키면 해내는 AI"로의 전환입니다.

### 3. 멀티모달의 일상화

텍스트, 이미지, 영상, 음성, 코드를 넘나드는 멀티모달 능력이 이제 기본 사양이 되었습니다. GPT-5와 Gemini 3은 출시 시점부터 네이티브 멀티모달을 지원합니다. 사진을 찍어서 "이게 뭐야?"라고 물으면 설명해주고, "이 스타일로 영상 만들어줘"라고 하면 만들어줍니다.

### 4. AI 비용의 급격한 하락

GPT-4 대비 GPT-5.4의 API 가격은 약 1/20로 하락했습니다. 오픈소스 모델은 무료로 사용할 수 있습니다. 1년 전만 해도 "비싸서 못 쓴다"던 AI를 이제는 스타트업도, 개인 개발자도 부담 없이 활용할 수 있게 되었습니다. AI 도입의 경제적 장벽이 사실상 사라진 것입니다.

### 5. 바이브 코딩의 부상

AI와 대화하며 코드를 작성하는 "바이브 코딩(Vibe Coding)"이 2025년 최대 유행어가 되었습니다. 비개발자도 v0, Replit Agent 등을 통해 웹 앱을 만들 수 있게 되면서, "5만 달러와 6개월이 필요했던 제품 출시가 노트북 하나와 주말 하나면 가능한 시대"가 열렸습니다. 바이브 코딩의 구체적인 실습 가이드는 Ch5에서 다룹니다.

## 4.5 이 챕터의 핵심 정리

1. **AI 서비스는 이미 일상에 침투해 있다**: 검색, 코딩, 디자인, 영상, 업무 자동화 -- 거의 모든 업무 영역에 AI 서비스가 존재합니다.
2. **엔진-완성차 구조를 이해하라**: 대부분의 서비스는 GPT, Claude 등 범용 모델을 API로 호출하며, 점점 더 멀티모델 전략을 취하고 있습니다.
3. **비즈니스 규모가 폭발적이다**: OpenAI ARR $250억, Anthropic $190억, Cursor $20억 -- AI는 실험이 아니라 거대한 산업입니다.
4. **실패에서도 배워라**: Humane AI Pin과 Rabbit R1의 실패는 "기술보다 사용자 습관이 중요하다"는 교훈을 남겼습니다.
5. **다섯 가지 메가트렌드를 주시하라**: 오픈소스 반격, 에이전트 시대, 멀티모달 일상화, 비용 하락, 바이브 코딩 -- 이 흐름이 향후 AI 산업의 방향을 결정합니다.
