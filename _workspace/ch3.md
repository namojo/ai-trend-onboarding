# Ch3. AI 모델 전쟁

> **한줄 요약**: OpenAI, Anthropic, Google, Meta, DeepSeek -- 다섯 진영이 벌이는 치열한 AI 모델 경쟁의 전략과 무기를 낱낱이 해부한다.

2024년부터 2026년 초까지, AI 업계에서는 말 그대로 **전쟁**이 벌어지고 있습니다. 불과 25일 사이에 주요 4개 회사가 차세대 모델을 연달아 출시하는 이례적인 사건이 벌어졌고, 오픈소스 진영은 수백만 달러의 학습 비용만으로 수십억 달러를 쏟아부은 상용 모델에 필적하는 성능을 달성해 업계를 뒤흔들었습니다.

이 챕터에서는 각 진영의 전략과 모델 라인업을 분석하고, 이 전쟁의 판도를 바꾼 결정적 순간들을 짚어봅니다.

## 3.1 전쟁의 다섯 진영

상상해보세요. 거대한 체스판 위에 다섯 명의 플레이어가 앉아 있습니다. 각자 고유한 전략과 병기를 가지고 있고, 누구도 쉽게 물러서지 않습니다. 이것이 바로 2024~2026년 AI 모델 시장의 풍경입니다.

| 진영 | 대표 모델 | 핵심 전략 | 한마디 정의 |
|------|-----------|-----------|-------------|
| **OpenAI** | GPT 시리즈, o 시리즈 | 선점자 우위 + 추론 혁신 | "규칙을 만드는 자" |
| **Anthropic** | Claude 시리즈 | 안전성 + 코딩 특화 | "신중한 도전자" |
| **Google** | Gemini 시리즈 | 초장문 컨텍스트 + 네이티브 멀티모달 | "인프라의 거인" |

| **DeepSeek** | V3, R1 시리즈 | 극한 효율 + 오픈소스 | "파괴적 혁신가" |

## 3.2 OpenAI -- 규칙을 만드는 자

### 3.2.1 GPT 계열의 진화

ChatGPT로 AI 대중화의 문을 연 OpenAI는 단순히 "먼저 시작한 회사"가 아닙니다. 이들은 전쟁의 **규칙 자체를 바꿔왔습니다**. GPT 계열의 진화 과정을 보면 그 전략 변화가 뚜렷하게 드러납니다.

| 모델 | 출시일 | 주요 특징 | 파라미터 수 | 컨텍스트 윈도우 |
|------|--------|----------|------------|----------------|
| GPT-3.5 | 2022년 11월 | ChatGPT의 기반 모델, 대화형 AI 대중화 시작 | 비공개 (GPT-3은 175B) | 4K → 16K |
| GPT-4 | 2023년 3월 | 멀티모달(텍스트+이미지) 입력, 추론 능력 대폭 향상 | 비공개 (추정 ~1.7T MoE) | 8K → 32K → 128K |
| GPT-4 Turbo | 2023년 11월 | 속도 향상, 비용 절감, 128K 컨텍스트 | 비공개 | 128K |
| GPT-4o | 2024년 5월 | 텍스트+이미지+오디오 네이티브 멀티모달 | 비공개 | 128K |
| GPT-4.5 | 2025년 2월 | 패턴 인식 향상, 환각(hallucination) 감소 | 비공개 | 128K |
| GPT-4.1 시리즈 | 2025년 4월 | API 중심 최적화, 4.1/4.1-mini/4.1-nano 3종 | 비공개 | **1M** |
| GPT-5 | 2025년 하반기 | 멀티모달 통합, 추론 능력 대폭 향상 | 비공개 | 비공개 |
| GPT-5.2 | 2025년 12월 | 코드 특화(Codex), 리포 단위 추론 | 비공개 | 비공개 |
| GPT-5.4 | 2026년 3월 | 최신 플래그십, 추론+속도 균형 | 비공개 | 비공개 |

OpenAI의 전략은 네 단계로 요약됩니다:

1. **더 큰 모델** -- GPT-3에서 GPT-4로, 파라미터를 키워 성능을 끌어올리는 스케일링
2. **더 빠르고 효율적인 모델** -- Turbo, 4o로 같은 성능을 더 싸고 빠르게
3. **더 깊이 생각하는 모델** -- o1, o3로 추론 시간에 더 많이 생각하게 하는 패러다임 전환
4. **용도별 특화 모델** -- 4.1 시리즈(API 최적화), Codex(코드 특화)로 세분화

### 3.2.2 o 시리즈 -- "생각하는 AI"의 탄생

2024년 하반기, OpenAI는 모델을 키우는 것만으로는 한계가 있다는 것을 깨닫고 완전히 새로운 방향을 제시합니다. 답변을 내놓기 전에 **추가 연산 시간을 투입하여 깊이 사고하는** 추론 모델, o 시리즈의 등장입니다.

| 모델 | 출시일 | 주요 특징 | API 가격 (입력/출력, /1M 토큰) |
|------|--------|----------|-------------------------------|
| o1 | 2024년 12월 | 최초 추론 특화 모델, 수학/코딩 탁월 | $15 / $60 |
| o3 | 2025년 1월 | o1 후속, 더 빠르고 정확한 추론 | $10 / $40 |
| o3-mini | 2025년 1월 | 경량 추론, 비용 효율적 | $1.10 / $4.40 |
| o3 Pro | 2025년 상반기 | 최고 수준 추론, 프로 전용 | $150 / $600 |

o 시리즈가 사용하는 **체인 오브 소트(Chain of Thought)** 추론은 마치 수학 시험에서 풀이 과정을 적으며 푸는 것과 같습니다. 바로 답을 내놓는 대신, 단계별로 생각을 전개하며 더 정확한 결론에 도달합니다. 이 접근법은 이후 Google, Anthropic, DeepSeek 모두가 따라가게 되는 업계 표준이 됩니다.

## 3.3 Anthropic -- 신중한 도전자

만약 OpenAI가 "빠르게 움직이고 혁신하라"는 실리콘밸리의 정신을 대표한다면, Anthropic은 "안전하게, 그러나 확실하게"라는 철학을 내세운 도전자입니다. OpenAI 출신 연구자들이 세운 이 회사는 **Constitutional AI(헌법적 AI)**라는 독자적 안전 기술을 앞세워 차별화에 성공했습니다.

| 모델 | 출시일 | 주요 특징 | 컨텍스트 윈도우 |
|------|--------|----------|----------------|
| Claude 1 | 2023년 3월 | Anthropic의 첫 상용 모델, Constitutional AI 적용 | 9K → 100K |
| Claude 2 | 2023년 7월 | 성능 대폭 향상, 200K 컨텍스트 지원 | 200K |
| Claude 3 (Opus/Sonnet/Haiku) | 2024년 3월 | 3단계 티어 체계 도입, 멀티모달(이미지) 입력 | 200K |
| Claude 3.5 Sonnet | 2024년 6월 | Opus급 성능을 Sonnet 비용으로, 코딩 능력 급상승 | 200K |
| Claude 3.7 Sonnet | 2025년 2월 | 하이브리드 추론 모델(확장된 사고 기능) | 200K |
| Claude 4 (Opus 4/Sonnet 4) | 2025년 5월 | 차세대 아키텍처, 에이전틱 코딩 능력 강화 | 200K |
| Claude Opus 4.5 | 2025년 11월 | 최고 수준 코딩/분석, $15/$75 | 200K |
| Claude Opus 4.6 | 2026년 2월 | 최신 플래그십 모델 | 200K |
| Claude Sonnet 4.6 | 2026년 2월 | 최신 효율형 모델, 코딩 벤치마크 최상위, $3/$15 | 200K |

### Anthropic의 비밀 무기: 티어 시스템

Claude의 독특한 전략은 **Opus / Sonnet / Haiku** 3단계 티어 시스템입니다. 마치 자동차 라인업처럼 용도에 맞는 선택지를 제공합니다:

- **Opus** = 프리미엄 세단: 최고 성능, 복잡한 분석과 추론에 최적
- **Sonnet** = 준중형 세단: 성능과 비용의 균형, 가장 인기 있는 선택
- **Haiku** = 경차: 빠르고 저렴, 대량 처리에 적합

특히 주목할 현상은 **세대 역전**입니다. Claude 3.5 Sonnet(중간 티어)이 이전 세대 Claude 3 Opus(최상위 티어)를 뛰어넘는 성능을 보여주었습니다. 작은 모델이 큰 모델을 이기는 이 패턴은 모델 효율성이 얼마나 빠르게 개선되고 있는지를 보여주는 증거입니다.

## 3.4 Google -- 인프라의 거인

Google은 AI 연구의 원조입니다. 2017년 트랜스포머 아키텍처를 발명한 곳이 바로 Google이니까요. 하지만 ChatGPT에 선수를 뺏긴 이후, 거인은 본격적으로 반격에 나섭니다.

| 모델 | 출시일 | 주요 특징 | 컨텍스트 윈도우 |
|------|--------|----------|----------------|
| Gemini 1.0 (Ultra/Pro/Nano) | 2023년 12월 | 네이티브 멀티모달, 3단계 티어 | 32K |
| Gemini 1.5 Pro | 2024년 2월 | **1M 토큰** 컨텍스트 윈도우의 혁신 | 1M → 2M |
| Gemini 1.5 Flash | 2024년 5월 | 경량 고속 모델, 비용 효율 극대화 | 1M |
| Gemini 2.0 Flash | 2025년 1월 | 새 기본 모델, 에이전틱 기능 강화 | 1M |
| Gemini 2.0 Pro | 2025년 2월 | 복잡한 작업용 고성능 모델 | 1M |
| Gemini 2.5 Pro | 2025년 3~6월 | "사고하는 모델", 체인 오브 소트 추론 | 1M |
| Gemini 2.5 Flash | 2025년 6~8월 | 추론 + 속도 + 저비용의 균형 | 1M |

### Google의 두 가지 초능력

**1. 초장문 컨텍스트**: Gemini 1.5 Pro가 100만 토큰 컨텍스트를 처음 선보였을 때, 업계는 충격에 빠졌습니다. 100만 토큰이면 대략 **책 7~8권 분량**을 한 번에 읽고 이해할 수 있다는 뜻입니다. 이 압도적인 컨텍스트 윈도우는 여전히 Google의 가장 강력한 차별점입니다.

**2. 가격 경쟁력**: Gemini 2.5 Flash의 API 가격은 입력 $0.30/1M 토큰, 출력 $2.50/1M 토큰으로, 경쟁사 대비 파격적으로 저렴합니다. Google은 자체 TPU 인프라를 통해 이러한 가격을 유지할 수 있습니다.

## 3.5 오픈소스 진영 -- 판도를 뒤흔든 도전자들

### 3.5.1 Meta의 LLaMA -- 개방의 전도사

2023년 2월, Meta가 LLaMA를 공개한 순간은 AI 역사의 분기점이었습니다. "왜 AI 모델을 공개하냐"는 질문에 Meta의 답은 명확했습니다 -- **생태계를 장악하기 위해서**.

| 모델 | 출시일 | 파라미터 수 | 주요 특징 |
|------|--------|------------|----------|
| LLaMA 1 | 2023년 2월 | 7B~65B | 오픈소스 LLM 붐의 시작 |
| LLaMA 2 | 2023년 7월 | 7B~70B | 상업적 사용 허가, RLHF 적용 |
| LLaMA 3 | 2024년 4월 | 8B~70B | 성능 대폭 향상, 15T 토큰 학습 |
| LLaMA 3.1 | 2024년 7월 | 8B/70B/**405B** | 최초의 오픈소스 400B+ 모델, 128K 컨텍스트 |
| LLaMA 4 Scout | 2025년 4월 | **109B(17B 활성)** MoE | **10M 토큰** 컨텍스트, 16명의 전문가 |

LLaMA 4 Scout의 **10M 토큰 컨텍스트**는 Google Gemini의 1~2M을 훌쩍 넘어서는 수치입니다. MoE(Mixture of Experts) 아키텍처를 채택하여 전체 109B 파라미터 중 실제 추론에는 17B만 활성화되므로, 거대한 모델이면서도 효율적으로 구동됩니다.

### 3.5.2 DeepSeek -- 파괴적 혁신가

2025년 1월, 중국의 DeepSeek이 R1을 공개한 순간, AI 업계 전체가 술렁였습니다. 그 이유는 단 하나 -- **학습 비용 약 557만 달러($5.576M)**로 프론티어 급 성능을 달성했기 때문입니다. 수십억 달러를 투자한 빅테크 기업들은 경악했습니다.

| 모델 | 출시일 | 파라미터 수 | 주요 특징 |
|------|--------|------------|----------|
| DeepSeek-V3 | 2024년 12월 | **671B(37B 활성)** MoE | 극도로 낮은 학습 비용($5.5M), MoE + Multi-head Latent Attention |
| DeepSeek-R1 | 2025년 1월 | 671B | 오픈소스 추론 모델의 시작, 체인 오브 소트, MIT 라이선스 |
| DeepSeek-V3.1 | 2025년 8월 | 671B(37B 활성) | V3 + R1 하이브리드, 128K 컨텍스트 |
| DeepSeek-V3.2 | 2025년 12월 | **685B** | MIT 라이선스, 128K 컨텍스트, 최강 오픈소스 모델 |

DeepSeek의 충격은 **"돈이 많아야 AI를 만든다"는 공식을 깨뜨린 것**입니다. 순수 강화학습(RL) 기반으로 추론 능력을 학습시킨 R1은 OpenAI o1에 필적하는 성능을 보여주면서, MIT 라이선스로 누구나 자유롭게 사용할 수 있도록 공개되었습니다.

## 3.7 전쟁의 승자는 누구인가

이 전쟁에서 확실한 것은 하나입니다 -- **단 하나의 승자는 없다**는 것.

### 각 진영별 승리 영역

- **OpenAI**: 범용 AI 시장의 선점자 우위 유지. 추론 모델(o 시리즈)이라는 새로운 카테고리 개척
- **Anthropic**: 코딩과 안전성에서 독보적 위치 확보. Claude Sonnet 4.6은 코딩 벤치마크 최상위
- **Google**: 초장문 컨텍스트와 가격 경쟁력으로 엔터프라이즈 시장 공략. 자체 TPU 인프라의 강점
- **Meta**: 오픈소스 생태계를 장악하여 AI 인프라의 표준을 만들어가는 중
- **DeepSeek**: "효율의 극한"을 증명하여 AI 민주화를 가속. 557만 달러로 프론티어 모델 학습이 가능함을 입증

### 2025~2026년의 핵심 트렌드

1. **추론 모델의 부상**: OpenAI o1을 시작으로, 모든 주요 진영이 "생각하는 모델"을 내놓고 있습니다. 단순히 빠르게 답하는 것이 아니라, 깊이 사고하여 더 정확한 답을 내는 방향으로 진화 중입니다.

2. **MoE 아키텍처의 대세화**: DeepSeek, LLaMA, Mixtral 등 오픈소스 진영을 중심으로 Mixture of Experts 구조가 표준이 되고 있습니다. 거대한 전체 파라미터 중 일부만 활성화하여 효율성을 극대화합니다.

3. **오픈소스의 역습**: 2025년 후반부터 오픈소스 모델이 상용 모델과 벤치마크에서 대등한 성능을 보이기 시작했습니다. 이는 AI 접근성의 민주화를 의미합니다.

4. **용도별 세분화**: 범용 모델 하나로 모든 것을 해결하던 시대에서, 추론용(o3), 코딩용(Codex, Claude Code), 경량용(Flash, Haiku, nano)으로 세분화되는 추세입니다.

5. **가격 전쟁 심화**: GPT-4 출시 당시 고가였던 API 가격이 1년 만에 수십 배 하락했고, 오픈소스 모델은 아예 무료입니다. AI 활용의 경제적 장벽이 급격히 낮아지고 있습니다.
