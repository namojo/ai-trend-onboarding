# 부록. AI 용어 사전

> 본 용어 사전은 교육 자료 전체에서 등장하는 AI 관련 전문 용어를 가나다순으로 정리한 것입니다.
> 각 용어의 영문 원어, 간단한 설명, 그리고 본문 참조 위치를 함께 제공합니다.

---

## ㄱ

| 용어 | 영문 | 설명 |
|:---|:---|:---|
| **거대언어모델** | LLM (Large Language Model) | 방대한 텍스트 데이터로 학습하여 인간의 언어를 이해하고 생성하는 AI 모델. GPT, Claude, Gemini 등이 대표적. (제4장 1절) |
| **강화학습** | Reinforcement Learning (RL) | AI가 시행착오를 통해 보상을 최대화하는 행동을 학습하는 방식. RLHF의 기반 기술. (제4장 3절) |
| **검색 증강 생성** | RAG (Retrieval-Augmented Generation) | AI가 답변 생성 전에 외부 문서를 검색하여 참고함으로써 정확도를 높이는 기술. "오픈북 시험"과 유사. (제4장 6절) |
| **가중치** | Weight | AI 모델 내부에서 학습된 연결의 강도를 나타내는 숫자. 매개변수(Parameter)와 유사한 개념. (제4장 2절) |

## ㄴ

| 용어 | 영문 | 설명 |
|:---|:---|:---|
| **노코드** | No-code | 프로그래밍 지식 없이 소프트웨어를 만들 수 있는 방식이나 도구. (제3장 5절) |

## ㄷ

| 용어 | 영문 | 설명 |
|:---|:---|:---|
| **다중 헤드 어텐션** | Multi-Head Attention | 여러 개의 Attention을 병렬로 실행하여 문법, 의미, 위치 등 다양한 관계를 동시에 파악하는 기법. (제4장 2절) |
| **디지털 전환** | DX (Digital Transformation) | 기업의 전략, 운영, 문화를 디지털 기술 기반으로 근본적으로 변화시키는 과정. AI 전환(AX)의 선행 개념. (제3장 3절) |

## ㅁ

| 용어 | 영문 | 설명 |
|:---|:---|:---|
| **매개변수** | Parameter | AI 모델이 학습 과정에서 조정하는 내부 값. 모델의 크기를 나타내는 지표로 사용. GPT-4는 수천억 개의 매개변수를 가짐. (제4장 1절) |
| **멀티모달** | Multimodal | 텍스트, 이미지, 음성, 영상 등 여러 종류의 데이터를 동시에 이해하고 처리하는 AI의 능력. GPT-4o, Gemini 등이 대표적. (제1장 3절, 제2장 2절) |
| **멀티모델 전략** | Multi-model Strategy | 하나의 AI 모델에 의존하지 않고, 업무 유형에 따라 복수의 AI 모델을 조합하여 사용하는 전략. (제1장 3절) |

## ㅂ

| 용어 | 영문 | 설명 |
|:---|:---|:---|
| **바이브 코딩** | Vibe Coding | AI에게 자연어(일상 언어)로 지시하여 프로그램을 만드는 방식. 비개발자도 간단한 프로그램 제작 가능. (제2장 6절) |
| **범용 모델** | Foundation Model | AI 서비스의 핵심 엔진이 되는 거대 AI 모델. OpenAI의 GPT, Anthropic의 Claude 등. "자동차의 엔진"에 비유. (제2장 1절) |
| **벡터** | Vector | 숫자들의 나열로, 임베딩에서 단어의 의미를 수학적으로 표현하는 데 사용. (제4장 2절) |
| **벤치마크** | Benchmark | AI 모델의 성능을 객관적으로 측정하는 표준 테스트. SWE-Bench(코딩), ARC-AGI(추론), AIME(수학) 등이 대표적. (제1장 2절) |
| **비판적 사고력** | Critical Thinking | AI 결과물의 정확성, 논리성, 맥락 적합성을 검증하고 판단하는 능력. AI 시대 가장 중요한 역량. (제5장 2절) |

## ㅅ

| 용어 | 영문 | 설명 |
|:---|:---|:---|
| **사전훈련** | Pre-training | AI 모델이 방대한 텍스트 데이터를 학습하여 범용적 언어 능력을 갖추는 첫 번째 학습 단계. (제4장 3절) |
| **생성형 AI** | Generative AI | 텍스트, 이미지, 영상, 코드 등 새로운 콘텐츠를 만들어내는 인공지능 기술의 총칭. (제1장 1절) |
| **소규모 언어모델** | sLLM (small Large Language Model) | 특정 업무나 산업에 특화된 소규모 언어모델. 비용 효율적이며 기업 환경에 맞게 배포 가능. (제1장 3절) |

## ㅇ

| 용어 | 영문 | 설명 |
|:---|:---|:---|
| **에이전트** | Agent (AI Agent) | 사람의 개입 없이 목표를 설정하고, 도구를 선택하며, 여러 단계의 작업을 자율적으로 수행하는 AI 시스템. (제2장 3절, 제3장 2절) |
| **에이전틱 AI** | Agentic AI | AI가 스스로 계획을 세우고, 도구를 선택하며, 다단계 작업을 자율 수행하는 AI 패러다임. 2025~2026년 핵심 트렌드. (제3장 2절) |
| **연간 반복 매출** | ARR (Annual Recurring Revenue) | SaaS 기업의 핵심 성과 지표. 구독 기반 연간 매출. (제2장 3절) |
| **영업이익** | EBIT (Earnings Before Interest and Taxes) | 이자와 세금 차감 전 영업 이익. 기업의 핵심 수익성을 나타내는 지표. (제1장 1절) |
| **오픈소스** | Open Source | 소스 코드를 공개하여 누구나 사용, 수정, 배포할 수 있는 소프트웨어 개발 방식. Meta의 LLaMA, DeepSeek 등이 대표적. (제2장 2절) |
| **온디바이스 AI** | On-device AI | 클라우드 서버가 아닌 스마트폰, PC 등 개인 기기에서 직접 작동하는 AI. Apple Intelligence가 대표적. (제1장 3절) |
| **응용 서비스** | Application Service | 범용 모델을 활용하여 특정 업무를 해결하는 최종 사용자 제품. Cursor, Perplexity 등. "자동차의 완성차"에 비유. (제2장 1절) |
| **임베딩** | Embedding | 단어나 문장의 의미를 숫자 벡터로 변환하는 기술. 의미가 유사한 단어는 비슷한 벡터값을 가짐. "단어에 GPS 좌표를 부여하는 것." (제4장 2절) |

## ㅈ

| 용어 | 영문 | 설명 |
|:---|:---|:---|
| **전문가 혼합** | MoE (Mixture of Experts) | 모델 내부에 여러 전문가 네트워크를 두고, 입력에 따라 일부만 활성화하는 효율적 아키텍처. DeepSeek V3, LLaMA 4 등에 사용. (제2장 2절, 제4장 4절) |
| **추론 모델** | Reasoning Model | 답변 전에 내부적으로 사고 과정(Chain of Thought)을 거치는 AI 모델. o1, o3, Claude 3.7 Sonnet Extended Thinking 등. (제2장 2절, 제4장 4절) |

## ㅊ

| 용어 | 영문 | 설명 |
|:---|:---|:---|
| **체인 오브 소트** | Chain of Thought (CoT) | AI가 문제를 풀 때 단계별로 사고 과정을 거치는 기법. 추론 모델의 핵심 기술. (제4장 4절) |

## ㅋ

| 용어 | 영문 | 설명 |
|:---|:---|:---|
| **컨텍스트 윈도우** | Context Window | AI가 한 번에 읽고 처리할 수 있는 텍스트의 최대 길이. "AI의 책상 크기"에 비유. 2026년 현재 최대 100만 토큰. (제1장 2절, 제4장 5절) |
| **컴플라이언스** | Compliance | 법률, 규정, 사내 정책 등을 준수하는 것. AI 시대에는 EU AI Act 등 AI 규제 준수가 중요. (제1장 2절) |

## ㅌ

| 용어 | 영문 | 설명 |
|:---|:---|:---|
| **토큰** | Token | AI가 텍스트를 처리하는 최소 단위. 영어 기준 약 0.75단어 = 1토큰, 한국어 기준 약 1글자 = 1~2토큰. (제1장 4절, 제4장 2절) |
| **토큰화** | Tokenization | 텍스트를 AI가 처리할 수 있는 토큰 단위로 분해하는 과정. "문장을 레고 블록으로 분해하는 것." (제4장 2절) |
| **트랜스포머** | Transformer | 2017년 Google이 발표한 AI 모델 구조. 현대 거의 모든 LLM의 기반. GPT의 'T'가 Transformer. (제4장 2절) |
| **Temperature** | Temperature | AI의 답변 다양성을 조절하는 설정값. 낮으면 보수적, 높으면 창의적인 답변 생성. (제4장 1절) |

## ㅍ

| 용어 | 영문 | 설명 |
|:---|:---|:---|
| **파인튜닝** | Fine-tuning | 범용 AI 모델을 특정 업무나 산업에 맞게 추가 학습시키는 과정. "대학원 진학"에 비유. (제4장 3절) |
| **폐쇄형 모델** | Closed Source / Proprietary | 소스 코드를 공개하지 않는 AI 모델. OpenAI의 GPT, Anthropic의 Claude 등. 오픈소스의 반대 개념. (제2장 2절) |
| **프롬프트** | Prompt | AI에게 보내는 지시문이나 질문. 프롬프트의 품질이 AI 결과물의 품질을 크게 좌우함. (제4장 5절, 제5장 2절) |
| **프롬프트 리터러시** | Prompt Literacy | AI에게 원하는 결과를 얻기 위해 효과적으로 질문하고 지시하는 능력. AI 시대 핵심 역량 중 하나. (제5장 2절) |

## ㅎ

| 용어 | 영문 | 설명 |
|:---|:---|:---|
| **환각** | Hallucination | AI가 사실이 아닌 정보를 마치 사실인 것처럼 자신 있게 생성하는 현상. LLM의 확률적 생성 원리에서 비롯되는 구조적 한계. (제1장 1절, 제4장 6절) |
| **확장 사고** | Extended Thinking | Claude 3.7 Sonnet 이후 도입된 기능으로, AI가 답변 전에 내부 사고 과정을 투명하게 보여주는 추론 방식. (제1장 2절, 제4장 4절) |
| **휴먼 인 더 루프** | Human-in-the-loop (HITL) | AI 의사결정 과정에 사람의 감독과 개입을 반드시 포함하는 원칙. 가장 확실한 환각 방지 수단. (제5장 3절) |

## A-Z (알파벳순 영문 약어)

| 약어 | 풀네임 | 설명 |
|:---|:---|:---|
| **AI** | Artificial Intelligence | 인공지능. 인간의 학습, 추론, 인지 능력을 모방하는 기술의 총칭. |
| **AIME** | American Invitational Mathematics Examination | 미국 수학 올림피아드 예선. AI 수학 추론 능력의 벤치마크로 활용. |
| **API** | Application Programming Interface | 프로그램 간 통신을 위한 인터페이스 규약. AI 모델을 다른 서비스에서 활용할 때 사용. |
| **ARC-AGI** | Abstraction and Reasoning Corpus for AGI | AI의 추상적 추론 능력을 측정하는 벤치마크. |
| **AX** | AI Transformation | AI 전환. 기업의 전략, 운영, 문화를 AI 기반으로 변화시키는 과정. DX(디지털 전환)의 차세대 개념. |
| **BPE** | Byte Pair Encoding | GPT 계열에서 사용하는 토큰화 방식. 자주 등장하는 문자 쌍을 하나의 토큰으로 합치는 알고리즘. |
| **CAGR** | Compound Annual Growth Rate | 연평균 성장률. 시장 규모의 성장 속도를 나타내는 지표. |
| **CRAFT** | Context, Role, Action, Format, Tone | 프롬프트 작성 프레임워크. 맥락, 역할, 행동, 형식, 톤의 5가지 요소. (제5장 2절) |
| **CSAT** | Customer Satisfaction Score | 고객 만족도 점수. 고객 서비스 품질을 측정하는 지표. |
| **DX** | Digital Transformation | 디지털 전환. 기업의 전략과 운영을 디지털 기술 기반으로 변화시키는 과정. |
| **EU AI Act** | European Union Artificial Intelligence Act | EU의 포괄적 AI 규제법. 2024년 8월 발효. 세계 최초의 포괄적 AI 규제 체계. |
| **GenAI** | Generative AI | 생성형 AI의 영문 약칭. |
| **GPT** | Generative Pre-trained Transformer | OpenAI의 거대언어모델 시리즈명. "사전훈련된 생성형 트랜스포머"의 약어. |
| **HITL** | Human-in-the-loop | 사람 참여 원칙. AI 의사결정에 사람의 감독을 반드시 포함하는 설계 원칙. |
| **IDE** | Integrated Development Environment | 통합 개발 환경. 코드 편집, 실행, 디버깅을 하나의 프로그램에서 수행하는 개발 도구. |
| **JD** | Job Description | 채용 공고의 직무 기술서. |
| **LLM** | Large Language Model | 거대언어모델. 방대한 텍스트를 학습하여 언어를 이해하고 생성하는 AI 모델. |
| **MAU** | Monthly Active Users | 월간 활성 사용자 수. 서비스의 이용 규모를 나타내는 지표. |
| **MoE** | Mixture of Experts | 전문가 혼합. 모델의 일부 전문가만 활성화하여 효율적으로 운영하는 아키텍처. |
| **RAG** | Retrieval-Augmented Generation | 검색 증강 생성. 외부 문서를 검색하여 AI 답변의 정확도를 높이는 기술. |
| **RLHF** | Reinforcement Learning from Human Feedback | 인간 피드백 기반 강화학습. 사람의 선호도를 반영하여 AI를 조정하는 학습 방법. (제4장 3절) |
| **ROI** | Return on Investment | 투자 수익률. AI 투자의 경제적 효과를 측정하는 지표. |
| **SaaS** | Software as a Service | 서비스형 소프트웨어. 인터넷을 통해 구독형으로 제공되는 소프트웨어. |
| **SEO** | Search Engine Optimization | 검색 엔진 최적화. AI 검색의 등장으로 전략적 변화가 필요한 영역. |
| **sLLM** | small Large Language Model | 소규모 언어모델. 특정 업무에 특화되어 비용 효율적인 언어모델. |
| **SWE-Bench** | Software Engineering Benchmark | AI의 소프트웨어 엔지니어링 능력을 측정하는 벤치마크. 실제 GitHub 이슈를 해결하는 방식으로 평가. |
| **TTS** | Text-to-Speech | 텍스트를 음성으로 변환하는 기술. ElevenLabs 등이 대표적. |
| **VOC** | Voice of Customer | 고객의 소리. 고객 피드백을 수집하고 분석하는 활동. |

*본 용어 사전은 2026년 3월 18일 기준으로 작성되었습니다. AI 분야는 새로운 용어가 빠르게 등장하므로, 정기적으로 업데이트할 것을 권장합니다.*
